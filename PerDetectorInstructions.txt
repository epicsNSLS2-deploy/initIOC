This file contains instructions for deploying IOCs for each detector.
Any detector not listed here is not guaranteed to be able to successfully generated.

For each you may edit the CONFIGURE file first, and then add the -p or -g flags to the ones below.

ADSimDetector
-------------
./initIOCs.py -c

-----------
./initIOCs.py

ADPointGrey
-----------
